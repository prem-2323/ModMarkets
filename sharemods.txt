The key authenticates your requests and links them to your account. How to get your key Request API access by contacting support Once enabled, go to My Account Click Generate API Key — your key will appear in a URL ready to use Base URL https://sharemods.com/api/ Response format All endpoints return JSON: { "status": 200, "msg": "OK", "result": { ... }, "server_time": "2026-04-24 13:00:00" } Status codes 200 Success 400 Bad request — missing or invalid parameters 403 Forbidden — invalid key or access denied 404 Not found 429 Too many requests — rate limit hit 451 Unavailable — DMCA takedown Rate limiting The API allows 100 requests per minute per IP address. Exceeding this returns HTTP 429 with a 1-hour block. Design your scripts to batch calls (e.g. comma-separated file codes) and add short delays between requests. Account Account info Returns basic information about your account — email, balance, premium status and storage usage. Request $ curl 'https://sharemods.com/api/account/info?key=key' Response { "status": 200, "msg": "OK", "result": { "email": "you@domain.com", "balance": "0.04900", "premium_expire": "2026-11-14 14:35:50", "storage_used": "7641143835", "storage_left": "inf" }, "server_time": "2026-04-24 13:00:00" } Response fields email — your account email balance — current balance premium_expire — premium expiry, or null if not premium storage_used — used storage in bytes storage_left — remaining storage in bytes, or "inf" if unlimited Upload Step 1 — get upload server URL Get the current upload URL and session ID. The upload server rarely changes, but it can be moved at any time without notice. We recommend fetching a fresh URL periodically (or before each upload run) — don't hardcode it. Request $ curl 'https://sharemods.com/api/upload/server?key=key' Response { "status": 200, "msg": "OK", "result": "https://xxxx.sharemods.com/cgi-bin/upload.cgi", "sess_id": "YOUR_SESSION_ID", "server_time": "2026-04-24 13:00:00" } Save both result (upload URL) and sess_id — you'll need them for Step 2. Step 2 — upload a file POST your file to the upload URL from Step 1 using multipart/form-data. Request $ curl -F "sess_id=SESS_ID" \ -F "utype=reg" \ -F "file_0=@path/to/file.zip" \ UPLOAD_URL Fields sess_id required — from Step 1 file_0 required — the file itself Response [ { "file_code": "b578rni0e1ka", "file_status": "OK" } ] File URL after upload https://sharemods.com/b578rni0e1ka Remote URL upload Queue a file to be downloaded from a remote URL. Our server will fetch it in the background. Only http:// and https:// URLs are accepted. Request $ curl 'https://sharemods.com/api/upload/url?key=key&url=https://example.com/file.zip&folder=0' Parameters key required — your API key url required — remote URL (http/https only) folder optional — folder ID to upload into (default: 0 = root) Response { "status": 200, "msg": "OK", "result": { "filecode": "b578rni0e1ka" }, "server_time": "2026-04-24 13:00:00" } The response is returned immediately, but file download happens asynchronously. Use /api/file/info to check whether the file has finished downloading. File management File info (single or batch) Get info about one or more files by file code. FAQ How do I get API access? API access is granted by request. Contact support and briefly describe how you plan to use the API (personal automation, integration project, mirror tool, etc.). Once approved, the Generate API Key button will appear in your My Account page. Does my API key expire? No. Your API key doesn't expire automatically. It remains valid until you regenerate it in your account page, or API access is revoked. Can I delete files through the API? No. File deletion is only available through the web interface. This is intentional — to prevent accidental data loss through automated scripts. Does /api/file/list return all files on the site? No. /api/file/list only returns files owned by your account. To check the status of any file by its code — regardless of the owner — use /api/file/info. What should I do if I hit the rate limit (HTTP 429)? Wait and retry after 1 hour. To avoid rate limits: Use batch operations — file_code=a,b,c for file/info and file/set_folder Add 1–2 second delays between requests in your scripts Avoid polling loops — the API is not designed for real-time monitoring My API key leaked. What should I do? Go to My Account and click Regenerate key. Your old key will immediately stop working. Update any integrations with the new key. Can I use the API to mirror files from another site? Yes — that's what the /api/upload/url endpoint is designed for.
